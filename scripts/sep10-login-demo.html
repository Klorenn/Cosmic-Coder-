SEP-10 Login (Freighter + backend)

Abre la consola del navegador (F12 → Consola) y ejecuta:

runSep10Login()

Requisitos: extensión Freighter instalada y backend con /auth/challenge y /auth/token en marcha.

Para que el módulo Freighter cargue, sirve el proyecto (npm run dev) y abre esta página desde la URL del servidor (ej. http://localhost:5173/scripts/sep10-login-demo.html).
